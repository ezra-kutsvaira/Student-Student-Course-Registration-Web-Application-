# User Guide

## Overview
This guide explains how users interact with the application.

## User Roles
The system supports two roles:

- `ADMIN`
- `STUDENT`

## Student Actions
A student can:

- log in,
- view available courses,
- register for courses,
- view registered courses.

## Admin Actions
An admin can:

- log in,
- add courses,
- update courses,
- delete courses,
- view all courses.

## Typical Student Workflow
1. Log in to the system.
2. Open available courses.
3. Select a course to register.
4. View registered courses.

## Typical Admin Workflow
1. Log in to the system.
2. Open course management.
3. Add, edit, or delete courses.
4. Review all course records.
